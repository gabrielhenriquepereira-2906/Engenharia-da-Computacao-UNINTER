## 📂 Trilha de Aprendizado / Learning Path

### 🔹 Semestre 01 (Em curso)
* **Lógica de Programação e Algoritmos:** Desenvolvimento de algoritmos estruturados e resolução de problemas utilizando **Python**.
* **Matemática Elementar:** Estudo de conjuntos numéricos, funções e logaritmos, estabelecendo a base analítica necessária para o cálculo de engenharia.
* **O Engenheiro de Computação e a Profissão:** Estudo das normas técnicas, ética e impacto das tecnologias de ponta.

A grade curricular do Bacharelado foi estruturada para suportar as seguintes áreas de especialização:
* **Sistemas de Baixo Nível:** Microprocessadores, Sistemas Embarcados e Eletrônica Digital.
* **Inteligência e Dados:** Inteligência Artificial Aplicada, Estrutura de Dados e Banco de Dados.
* **Infraestrutura e Redes:** Sistemas Operacionais, Redes de Computadores e Computação em Nuvem.
* **Automação:** Robótica Autônoma e Móvel, Visão Computacional e Internet das Coisas (IoT).
