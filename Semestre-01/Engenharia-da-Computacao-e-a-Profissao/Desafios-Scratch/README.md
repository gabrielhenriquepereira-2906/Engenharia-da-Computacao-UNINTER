# Engenharia da Computação e a Profissão - Práticas com Scratch

Este diretório armazena as atividades práticas desenvolvidas com a ferramenta Scratch para a disciplina introdutória de Engenharia de Computação.

## Objetivo / Objective
Validar conceitos fundamentais de algoritmos, como eventos, loops, condicionais e lógica de estados. 
*(To validate fundamental algorithm concepts, such as events, loops, conditionals, and state logic.)*

---

## Exercício 01: Saída de Dados e Mudança de Estado
* **Descrição:** Implementação de um fluxo básico iniciado por evento de gatilho. O algoritmo realiza o output de texto e altera a propriedade visual (fantasia) do ator.
* **Conceitos:** Event-driven programming, Delays e State Switching.

---

## Exercício 02: Movimentação Iterativa e Sensores de Colisão
* **Descrição:** Desenvolvimento de um algoritmo de controle de movimento automatizado utilizando loops infinitos. O sistema monitora continuamente a posição do ator através de sensores, executando um reset para a origem (X:0, Y:0) ao detectar contato com a borda.
* **Conceitos:** Loops Iterativos (Forever), Lógica Condicional (If/Then) e Manipulação de Eixos Cartesianos.

![Código Exercício 02](./Bloco%20Scratch%20Exercício%202.png)

---

## Exercício 03: Simulação de Jogo com Lógica de Pontuação
* **Descrição:** Desenvolvimento de uma lógica de jogo completa para dois jogadores. O sistema utiliza variáveis globais para contagem de score, detecção de colisão entre múltiplos objetos e aleatoriedade para simular uma partida de futebol.
* **Conceitos Técnicos:**
    * **Variáveis de Estado:** Gerenciamento de `pontos1` e `pontos2`.
    * **Lógica Condicional Aninhada:** Estruturas `if-then-else` para validação de gols e direção da bola.
    * **Sistemas de Feedback:** Output visual ("Ponto!") condicionado ao estado das variáveis.
    * **Randomness:** Implementação de posição aleatória para garantir a dinâmica do sistema.
