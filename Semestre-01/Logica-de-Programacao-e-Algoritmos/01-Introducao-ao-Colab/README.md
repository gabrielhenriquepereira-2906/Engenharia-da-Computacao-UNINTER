# **Introdução ao Google Colab | Introduction to Google Colab**
Este diretório contém os exercícios iniciais desenvolvidos em ambiente de nuvem utilizando o **Google Colab**. O objetivo desta etapa foi familiarizar-se com a execução de scripts Python em células, manipulação de bibliotecas básicas e visualização de dados inicial.

*This directory contains initial exercises developed in a cloud environment using **Google Colab**. The objective was to become familiar with executing Python scripts in cells, basic library manipulation, and initial data visualization.*
